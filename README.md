EcoStat
AI-Driven Sustainability Rating Platform

EcoStat is a full-stack AI-powered web platform that evaluates sustainability performance of institutions and communities using structured environmental metrics and generates data-driven sustainability scores and reports.

🚀 Engineering Objective

-To design a scalable, modular system that:

-Collects structured environmental input data

-Processes it using a weighted AI-based scoring engine

-Generates composite sustainability ratings

-Stores and verifies results securely

🏗 System Architecture
1️⃣ Frontend (Angular)

-Dynamic form-based data collection

-Real-time validation

-API integration with backend services

-Dashboard for score visualization

2️⃣ Backend (Node.js + REST APIs)

-Modular controller-service architecture

-Input validation & preprocessing

-Scoring engine integration

-Report generation module

-Secure API endpoints

3️⃣ AI Scoring Engine (Python)

-Weighted multi-parameter scoring model

-Feature normalization

-Composite sustainability index generation

-Extensible for ML-based predictive analytics

4️⃣ Database Layer

-MongoDB/MySQL for structured environmental data

-Indexed score storage for efficient retrieval

5️⃣ Blockchain Verification (Concept Module)

-Smart contract-based score hash storage

-Tamper-proof verification of sustainability reports

📊 Core Evaluation Parameters

-Water Management Efficiency

-Waste Processing & Recycling Rate

-Energy Consumption Patterns

-Indoor Air Quality Metrics

-Carbon Emission Levels

-Each parameter contributes to a weighted sustainability index.

🛠 Tech Stack

Frontend: Angular, TypeScript, HTML, CSS

Backend: Node.js, Express.js

AI Module: Python

Database: MongoDB / MySQL

Version Control: Git & GitHub

🔍 Key Engineering Highlights

-RESTful API design

-Separation of concerns (frontend, backend, AI module)

-Scalable architecture

-Real-world domain modeling

-Modular scoring logic (easily extendable)
